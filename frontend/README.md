# 🎨 URL Shortener — Frontend

## 📌 Description
Frontend application for the URL shortener built with React providing user interface for link generation, tracking, and management.

## 🚀 Tech Stack
- React.js
- React Router
- Fetch API / Axios
- CSS
- Vite

## ✨ Features
- Create short URLs with validation
- Copy short URL with one click
- View all links with search and filter
- Track clicks and view link statistics
- Delete links from dashboard
- Responsive UI for all devices
- Toast notifications for success and errors

## 🔗 Integration
- Connected with Spring Boot backend APIs
- Handles API requests for link creation, retrieval, and deletion
- Displays real-time updates for link statistics
